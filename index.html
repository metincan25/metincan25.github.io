Güçlü Teknoloji Altyapısı

Inventra, modern ve güvenilir teknolojilerle inşa edilmiştir.

Backend Teknolojileri

C# & ASP.NET Core 8

Güçlü, performanslı ve modern RESTful API altyapısı

Entity Framework Core & MS SQL Server

Veritabanı işlemleri için ORM ve ilişkisel veritabanı

JWT (JSON Web Token)

Güvenli kimlik doğrulama sistemi

ClosedXML

Excel'e veri aktarımı için gelişmiş kütüphane

HTML5, CSS3, JavaScript (ES6+)

Bootstrap 5

Duyarlı ve modern tasarım için CSS framework

Chart.js

Grafiksel raporlamalar için JavaScript kütüphanesi

Cloudflare

DDoS koruması ve CDN hizmetleri
